- `tool_executor.py:250,257`: 丢弃 LLM 的 `[emotion:xxx]` 标签后重新检测
- 三套互相不一致的情绪映射表（EMOTION_MAP / _DESC_EMOTION_MAP / EMOTION_ALIASES）
