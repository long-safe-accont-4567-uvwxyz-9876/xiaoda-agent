# 📖 使用指南

## CLI 模式

```bash
python agent.py
```

### 常用命令
- `/help` - 显示帮助
- `/clear` - 清除对话历史
- `/memory` - 查看记忆
- `/tools` - 查看可用工具
- `/status` - 查看系统状态

## QQ Bot 模式

1. 在 QQ 开放平台创建 Bot
2. 配置 `.env` 中的 `APP_ID` 和 `APP_SECRET`
3. 运行 `python qq_bot_adapter.py`

## Web UI 模式

```bash
python web/app.py
```
